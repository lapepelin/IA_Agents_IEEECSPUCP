Sección 02
Agenda del Workshop
A lo largo de las sesiones, aprenderemos a construir sistemas de IA desde la base hasta despliegues complejos con herramientas.
Módulo 01

Fundamentos LLM

Diferencias arquitectónicas entre modelos (NVIDIA Nemotron vs Llama 3.3). Estructuración y manejo avanzado de prompts del sistema.

Módulo 02

Conexión con APIs

Uso de lenguajes como Python y Node.js para consumir Large Language Models utilizando SDKs estándar (compatibles con OpenAI).

Módulo 03

AI Agent Basic (Telegram)

Creación de un bot nativo de manera asíncrona. Implementación de memoria corta conversacional y manejo de configuraciones vía entorno `.env`.

Módulo 04

Persistencia de Datos

Mecanismos para almacenar historiales de conversaciones individualizadas en formato JSON, guardando el progreso de manera local.

Módulo 05

AI Agent Web

Construcción de una interfaz visual web minimalista empleando Vanilla JS en el Frontend y un servidor Express como proxy en el Backend.

Módulo 06

AI Agent Intermediate

Introducción al uso de Herramientas extensibles (Tool Calling), bases de datos y la integración de RAG para la lectura eficiente de archivos PDF.
